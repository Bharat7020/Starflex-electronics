# StarFlex Electronics

A full-stack e-commerce web application built using React.js, Context API, Node.js, and Express.js.

## Features

- User Authentication
- Product Catalog
- Product Details
- Shopping Cart
- Checkout Flow
- Responsive Design
- REST API
- JSON Data Storage

## Tech Stack

### Frontend

- React.js
- Context API
- JavaScript
- HTML5
- CSS3
- Vite

### Backend

- Node.js
- Express.js
- REST API

### Storage

- JSON

## Project Structure

```text
StarFlex-Electronics
├── frontend
└── backend
```

## Installation

### Clone

```bash
git clone https://github.com/Bharat7020/Starflex-electronics.git
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm start
```
